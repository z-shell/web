Zsh version 3.1.5 has now been released!

  • Globbing now supports the POSIX character classes, so that, for example, [[:upper:]_] now does what was traditionally expressed as [A-Z_].
  • The universal-argument ZLE function now behaves like the EMACS one, accepting an arbitrary numeric argument.
  • A bug in utmp handling that prevented compilation on HP-UX is fixed.
  • There are many bugfixes and other changes. For a more detailed list, see the ChangeLog file.
