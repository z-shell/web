About these Pages

These hypertext pages contain a collection of information concerning the Z-shell (ZSH), a freely available command interpreter for many UNIX systems. The information is provided as a series of links on the World Wide Web although many of the links are local now.

This project began with the HTML'ification of the ZSH Frequently Asked Questions (FAQ). This is now partially integrated with hypertext links to the ZSH manual pages and indices. Note, however, that this hypertext manual may be temporarily lagging behind the development version of ZSH.

If you find any additional references or find any errors, please take a moment to tell us about it.

The website is maintained in a Git repository. You can clone it with:

git clone git://git.code.sf.net/p/zsh/web zsh

You can also browse the repository online through SourceForge's Git viewer.
