Scripts

This page is dedicated to examples of zsh scripts.

  • The standard list distributed with zsh-3.0.5
    • zed: Edit small files with the command line editor
    • c2z: Convert csh aliases to zsh aliases/functions
    • multicomp: Completes all manner of files given prefixes for each path segment. e.g. s/z/s -> src/zsh-2.4/src
