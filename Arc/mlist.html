ZSH Mailing Lists

At this time, you can manage all your subscriptions at www.zsh.org/sympa.

Zsh has four mailing lists:

  1. An announce list.
    zsh-announce@zsh.org
    [Subscribe]
    • This is an announcements-only list; you will not be able to post to it.
  2. A users list.
    zsh-users@zsh.org
    [Subscribe]
    • This is a low-medium volume list with questions and information mainly about the driving of the shell.
  3. A workers/hackers list.
    zsh-workers@zsh.org
    [Subscribe]
    • A higher volume list with questions and stuff and things about the guts of the shell.
    • Subscribers to this list also receive all the messages from the users list.
  4. A security report list.
    zsh-security@zsh.org
    • This is not open for the public to subscribe to, but you can send email to it.
    • If you find a security problem in zsh itself, please mail this address.
To subscribe or unsubscribe, select the appropriate link above or send mail to:
SUBSCRIPTION ADDRESSES     UNSUBSCRIPTION ADDRESSES
zsh-announce-subscribe@zsh.org     zsh-announce-unsubscribe@zsh.org
zsh-users-subscribe@zsh.org     zsh-users-unsubscribe@zsh.org
zsh-workers-subscribe@zsh.org     zsh-workers-unsubscribe@zsh.org

You will receive a confirmation request. Reply to the request to authorise your subscription to (or unsubscription from) the mailing list. You must unsubscribe from the mailing list you subscribed to. If you don't know which list you are subscribed to, look in the headers of mailing list messages. The should be numerous pointers (e.g. Mailing-List: header)

N.B. This method may be different from that used when you subscribed

YOU ONLY NEED TO JOIN ONE OF THE MAILING LISTS AS THEY ARE NESTED.
All submissions to zsh-users are automatically forwarded to zsh-workers.

If you have problems subscribing/unsubscribing to any of the mailing lists, send mail to the mailing list administrator <zsh-announce-owner@zsh.org>

Archives

The mailing lists are archived; to find out how to use the automated mailing list responses send a message to one of the help addresses.

There is also a hypertext archive, maintained by zsh.org volunteers, available at https://www.zsh.org/mla/.
