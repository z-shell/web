ZSH Documentation

The Zsh manual is available in the following formats:

  • HTML, Browsable Hypertext
  • HTML (tarred and gzipped package, 540KiB)

  • PostScript (European A4 size paper, gzipped, 920KiB)
  • PostScript (US letter size paper, gzipped, 920KiB)
  • PDF (European A4 size paper, 2300KiB)
  • PDF (US letter size paper, 2200KiB)
  • DVI (gzipped, 650KiB)

  • INFO (tarred and gzipped package, 400KiB)
  • TexInfo source (gzipped, 380KiB)

As such, the manual includes the following indices:

  • Concept Index
  • Variables Index
  • Options Index
  • Functions Index
  • Editor Functions Index
  • Style and Tag Index

The documentation is prepared using Yodl which provides the conversion to troff man pages and to TeXinfo. The HTML version is derived from the TeXinfo version using texi2html.
